# ☕ Java CLI Projects

A collection of command-line projects built with **Core Java** for practice and learning.

## 📌 About

This repository contains small, focused CLI applications written in pure Java — no frameworks, no external libraries. Just core Java concepts in action.

## 🛠️ Tech Stack

- **Language:** Java (Core Java)
- **Build:** Manual (`javac`) or any IDE of your choice
- **JDK:** Java 8+

## 📁 Project Structure

```
java-cli-projects/
├── project-1/
│   ├── src/
│   └── README.md
├── project-2/
│   ├── src/
│   └── README.md
└── ...
```

## 🚀 How to Run

1. Navigate to a project folder
2. Compile the source file:
   ```bash
   javac src/Main.java
   ```
3. Run the program:
   ```bash
   java -cp src Main
   ```

## 💡 Topics Practiced

- OOP (Classes, Inheritance, Interfaces)
- Collections (ArrayList, HashMap, etc.)
- File I/O
- Exception Handling
- String manipulation
- Loops, conditionals, and control flow
